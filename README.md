# heart-disease-analysis
Projekt analizy czynników ryzyka chorób serca wykonany w języku R.

## Zakres analizy
- eksploracja danych (EDA)
- wizualizacja danych (ggplot2)
- testy statystyczne
- regresja liniowa

## Narzędzia
- R
- ggplot2
- dplyr

## Wnioski
Analiza wykazała zależność między wiekiem, ciśnieniem i cholesterolem a ryzykiem choroby, jednak wyniki odbiegają od obserwacji medycznych, co może wynikać ze specyfikacji użytych danych.

## Źródła
- Dane: Zbiór "Heart Disease Dataset" pobrany z platformy Kaggle:         https://www.kaggle.com/datasets/johnsmith88/heart-disease-dataset?resource=download
- Wsparcie AI: ChatGPT został wykorzystany do pomocy przy debugowaniu kodu wizualizacji oraz wstępnej redakcji opisu wniosków statystycznych.
