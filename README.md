# heart-disease-analysis
Projekt analizy czynników ryzyka chorób serca wykonany w języku R.

## Zakres analizy
- eksploracja danych (EDA)
- wizualizacja danych (ggplot2)
- testy statystyczne
- regresja logistyczna

## Narzędzia
- R
- ggplot2
- dplyr
- RMarkdown
  
## Wnioski
Analiza wykazała zależność między wiekiem, ciśnieniem i cholesterolem a ryzykiem choroby, jednak wyniki odbiegają od obserwacji medycznych, co może wynikać ze specyfiki użytych danych.

## Źródła
- Dane: Zbiór "Heart Disease Dataset" pobrany z platformy Kaggle:         https://www.kaggle.com/datasets/johnsmith88/heart-disease-dataset
- Wsparcie AI: ChatGPT został wykorzystany do pomocy przy debugowaniu kodu, wizualizacji oraz wstępnej redakcji opisów wniosków statystycznych.
